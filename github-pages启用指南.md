# GitHub Pages 启用指南

恭喜！你的代码已成功推送到 GitHub 仓库 `https://github.com/baiweny/airplane-game`。现在需要启用 GitHub Pages 来生成外网访问链接。

## 步骤 1：打开 GitHub 仓库

1. 访问 [GitHub](https://github.com/) 并登录你的账号
2. 打开你的仓库：https://github.com/baiweny/airplane-game

## 步骤 2：进入设置页面

1. 点击仓库页面顶部的 "Settings"（设置）选项卡

## 步骤 3：配置 GitHub Pages

1. 在左侧菜单中找到并点击 "Pages"
2. 在 "Source" 部分：
   - 选择 "Deploy from a branch"
   - 在 "Branch" 下拉菜单中选择 "master" 分支
   - 在 "Folder" 下拉菜单中选择 "/ (root)"
   - 点击 "Save"

## 步骤 4：获取外网链接

- 保存设置后，GitHub Pages 会开始部署你的网站
- 部署完成后，你会在 "GitHub Pages" 设置页面看到类似 "Your site is live at https://baiweny.github.io/airplane-game/" 的链接
- 这个链接就是你的游戏外网访问地址，复制并分享给他人即可

## 步骤 5：测试游戏

部署完成后，打开生成的外网链接，测试游戏是否能正常运行：
1. 检查游戏是否能正常加载
2. 测试 WASD 控制是否正常
3. 测试游戏功能是否完整
4. 测试在不同浏览器中的兼容性

## 注意事项

- **部署时间**：GitHub Pages 部署通常需要 1-2 分钟
- **自动更新**：每次向 GitHub 推送新代码后，GitHub Pages 会自动重新部署
- **自定义域名**：如果需要使用自定义域名，可以在 GitHub Pages 设置中配置
- **HTTPS 支持**：GitHub Pages 默认提供 HTTPS 支持

## 外网访问链接

启用 GitHub Pages 后，你的游戏将可以通过以下链接访问：
**https://baiweny.github.io/airplane-game/**

现在你可以将这个链接分享给你的朋友和家人，让他们也能体验你的游戏！
